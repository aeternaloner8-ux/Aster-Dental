# Aster Dental

Premium static website for a dental clinic.

## Pages

- `index.html` — main landing page
- `about.html` — clinic and team
- `portfolio.html` — services and prices
- `process.html` — treatment process and FAQ
- `reviews.html` — patient reviews
- `contacts.html` — contacts and appointment form

## Publish With GitHub Pages

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to **Pages**.
4. Set source to **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save.

The site will be available at:

`https://aeternaloner8-ux.github.io/aster-dental/`

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 4173
```

Then open:

`http://127.0.0.1:4173/index.html`
